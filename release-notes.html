INTERNAL INITIAL
2026.07.14

포함된 초기 기능

  • Windows 10/11 64비트용 설치 프로그램
  • 로스트아크 플레이 보조용 데스크톱 인터페이스
  • 게임 위 오버레이를 통한 정보 표시 기반
  • 시세 감시와 가격 알림을 위한 화면 구성
  • 쌀산기, 젬 가공, 일반 재련 등 도우미 메뉴 기반

배포 웹사이트

  • jloa.cloud 공식 다운로드 페이지 공개
  • Caddy를 이용한 HTTPS 적용
  • VirusTotal 검사 결과 링크 제공
  • SHA-256 체크섬 공개
  • 개인정보 처리방침과 광고 정책 페이지 추가

알려진 사항

  • 설치 파일에 공인 코드 서명이 적용되지 않았습니다.
  • Chrome에서 다운로드 빈도가 낮은 신규 파일 경고가 표시될 수 있습니다.
  • Windows에서 확인되지 않은 게시자 또는 SmartScreen 경고가 표시될 수 있습니다.
  • 회원가입, Premium 결제, 광고 제거, 클라우드 동기화는 아직 제공되지 않습니다.
  • 자동 업데이트 기능이 없으므로 새 버전은 공식 웹사이트에서 다시 받아야 할 수 있습니다.
