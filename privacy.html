현재 공개 버전의 기준

현재 JLOA는 회원가입, 로그인, 결제 또는 클라우드 동기화 기능을 제공하지 않으며, 앱의 설정·감시 항목·계산 결과를 JLOA 계정 서버에 저장하지 않습니다. 이후 해당 기능을 도입할 경우 수집 항목과 보유 기간을 먼저 고지합니다.

1. 적용 범위

본 방침은 JLOA가 운영하는 웹사이트 jloa.cloud와 Windows용 JLOA 데스크톱 애플리케이션에 적용됩니다. 사용자가 외부 사이트 또는 제3자 서비스로 이동한 이후에는 해당 사업자의 정책이 적용됩니다.

2. 처리하는 정보

구분 처리될 수 있는 정보 수집 방식
웹사이트 접속 IP 주소, 접속 시각, 요청 주소, 브라우저·운영체제 정보, 이전 페이지 주소 등 기술 정보 웹서버, 보안 장비 또는 호스팅 환경이 자동 생성
이메일 문의 이메일 주소, 표시 이름, 문의 내용 및 사용자가 첨부한 파일 사용자가 이메일로 직접 전송
데스크톱 앱 현재 버전에서는 계정 개인정보를 요구하지 않음 해당 없음
오류 진단 사용자가 직접 전달한 앱 버전, 오류 메시지, 로그, 시스템 환경 정보 사용자가 문의 과정에서 선택적으로 제공

JLOA는 주민등록번호, 계좌 비밀번호, 결제 카드 전체 번호와 같은 민감한 정보를 이메일이나 앱을 통해 요구하지 않습니다.

3. 처리 목적

  • 웹사이트와 설치 파일의 안정적인 제공
  • 오류 분석, 보안 위협 대응 및 비정상 요청 방지
  • 사용자 문의 접수와 답변
  • 서비스 품질 개선과 배포 문제 확인
  • 법령상 의무 이행 및 분쟁 대응

4. 보유 및 이용 기간

정보 기준 보유 기간
일반 웹 접속·보안 로그 보안과 장애 대응에 필요한 범위에서 최대 90일. 별도 저장하지 않는 로그는 서버 종료 또는 로그 순환 시 삭제
이메일 문의와 답변 기록 문의 처리 완료 후 최대 1년. 분쟁 또는 법적 의무가 있는 경우 필요한 기간 동안 별도 보관
사용자가 제공한 오류 로그·첨부파일 문제 해결 완료 후 30일 이내 삭제를 원칙으로 하며, 재현이 필요한 경우 사용자에게 추가 안내

처리 목적이 달성되거나 보유 기간이 끝난 정보는 복구가 어렵도록 삭제합니다.

5. 기기 안에 저장되는 정보

앱의 설정, 알림 조건, 캐시 또는 기능 사용에 필요한 데이터는 사용자의 PC에 저장될 수 있습니다. 이 정보는 기본적으로 JLOA 계정 서버로 전송되지 않으며, 앱 삭제만으로 일부 설정 폴더가 남을 수 있으므로 사용자는 앱 설정 또는 Windows의 앱 데이터 폴더에서 직접 삭제할 수 있습니다.

앱이 외부 게임 데이터·가격 정보 등을 가져오기 위해 인터넷에 접속하는 경우, 해당 API 또는 네트워크 제공자에게 IP 주소와 요청 정보가 전달될 수 있습니다.

6. 외부 서비스 및 제3자 처리

Google Fonts

웹사이트는 글꼴을 표시하기 위해 Google Fonts에 접속할 수 있습니다. 이 과정에서 사용자의 IP 주소와 브라우저 연결 정보가 Google에 전달될 수 있습니다. Google의 처리 방식은 Google 개인정보처리방침 에서 확인할 수 있습니다.

VirusTotal

사용자가 보안 검사 결과 링크를 직접 열면 VirusTotal 웹사이트로 이동합니다. 이동 이후의 접속 정보는 VirusTotal 운영자의 정책에 따라 처리됩니다.

광고 서비스

현재 공개 버전에는 제3자 맞춤형 광고가 활성화되어 있지 않습니다. 향후 웹사이트에 Google AdSense 또는 다른 광고 네트워크를 적용할 경우, 쿠키·개인 맞춤 광고·국외 처리 등 필요한 내용을 본 방침과 광고 정책에 반영하고 필요한 동의 절차를 제공합니다.

로그인·결제 기능

현재 로그인과 결제 기능은 제공되지 않습니다. 도입 시 인증 제공자, 결제대행사, 수집 항목, 위탁 또는 제3자 제공 여부, 국외 이전 여부와 보유 기간을 실제 적용 전에 공개합니다.

7. 이용자의 권리와 행사 방법

이용자는 자신과 관련된 개인정보의 열람, 정정, 삭제, 처리정지 또는 동의 철회를 요청할 수 있습니다. 현재 계정 시스템은 없으므로 주로 이메일 문의와 사용자가 직접 제공한 자료가 대상입니다.

요청은 JLOA 오픈카카오톡 문의 로 접수할 수 있으며, 본인 확인이 필요한 경우 요청 범위 내에서 최소한의 정보를 추가로 확인할 수 있습니다.

8. 개인정보 안전성 확보 조치

  • HTTPS를 통한 웹사이트 및 다운로드 전송 암호화
  • 불필요한 개인정보 수집 최소화
  • 서버와 운영 계정 접근 권한 제한
  • API 키·비밀번호 등 비밀정보의 소스코드 분리
  • 보안 사고 확인 시 관련 정보 보존과 원인 조사

9. 만 14세 미만 이용자

현재 JLOA는 계정 가입을 받지 않습니다. 향후 만 14세 미만 이용자의 개인정보를 수집해야 하는 기능을 제공하는 경우, 법령에서 요구하는 법정대리인 동의와 확인 절차를 적용합니다.

10. 개인정보 처리방침 변경

기능, 광고, 로그인, 결제 또는 분석 도구의 변경으로 개인정보 처리 내용이 달라지는 경우 시행 전에 웹사이트를 통해 변경 내용과 시행일을 알립니다. 이용자 권리에 중대한 영향을 주는 변경은 가능한 한 눈에 띄는 방식으로 안내합니다.
